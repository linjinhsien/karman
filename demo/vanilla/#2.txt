函式庫開發 - API 抽象層 #2

其中一個核心模組「驗證引擎 Validation Enigine」，會在接收到 Input/Payload 時，依照封裝 API 所定義的參數規則進行驗證，其中可接受的驗證規則有：

- Type: 以字串表示之型別，包含原始及物件型別，但型別在此驗證引擎中會有更詳細的定義。
- Constructor: 建構函式，將檢測參數是否為其實例。
- Function: 客製化驗證函式，因本身型別上與建構函式相同，需要另外使用 defineCustomValidator 創建。
- RegExp: 正則表達式。
- Descriptor: 描述參數最大、最小、或相等值，以及觀測屬性。
- RuleSet: 為上述所有規則的集合，又可分為聯集與交集。
